The What

Wikipedia Search Engine

This is a Wikipedia search app you can search anything in it or go to any random article it uses Wikipedia's API to fetch essential data for you. It has a minimal design.

Weather App

This application uses your device's location and shows weather around you, forecasting, windspeed, temperature(click to change), Timezone, coordinates it shows different color and icons according to weather around you.

Simon Game

This is a Simon Game made using Jquery, bootstrap, HTML5, CSS3. It has a strict mode and a normal mode you can restart any time and it holds your score.

Tic Tac Toe Game

This is a Tic TacToe Game made using Jquery, bootstrap, HTML5, CSS3. You can restart any time and choose your sign.

Twitch Streamers

This shows you channels currently broadcasting and channels that are offline and a list of selected channels. This uses Twitch tv API to fetch data.

Random Quote Machine

This is a Random Quote generator made using Jquery. It shows you random quotes that it pulls from forismatic API. It has a tweet button which allows you to tweet the quote instantly. The API uses Http so it is not functional on gthub pages and codepen.

JavaScript Calculator

This is a JavaScript calculator made using Jquery, bootstrap, HTML5, CSS3. It shows you a history of you calculations in secondary screen and has a cancel and clear button.
